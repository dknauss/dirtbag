Dirtbag
=======

Dirtbag is a small WordPress block theme for plain publishing, old-open-web habits, accessibility, comments, feeds, OPML, XFN, microformats, and web-safe colours.

It favours:

- semantic HTML
- core WordPress blocks
- visible links
- square buttons
- RSS
- comments
- OPML blogrolls
- no theme-authored frontend JavaScript in v1

Useful pages:

- /features/
- /colour-styles/
- /accessibility/
- /colophon/
- /now/
- /links/
- /feeds/

The truck is rusty. The links work.
