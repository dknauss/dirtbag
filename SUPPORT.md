# Support

Thanks for using Dirtbag.

## Where to get help

- **Bug reports:** Use the GitHub bug report issue form.
- **Compatibility or theme-review findings:** Use the GitHub compatibility report issue form.
- **Small improvements:** Use the GitHub enhancement request form.
- **Security vulnerabilities:** Use GitHub private vulnerability reporting. Do not open a public issue for security reports.

## Before opening an issue

Please include:

- WordPress version.
- PHP version.
- Dirtbag version or commit SHA.
- Browser and assistive technology details when relevant.
- Steps to reproduce in a clean WordPress install or WordPress Playground when possible.

Dirtbag intentionally avoids theme-authored front-end JavaScript, build steps, bundled web fonts, and remote assets. Requests that preserve those constraints are easiest to review.
